Getting started

Welcome

Hello there! Thanks for buying Biz Mart. Biz Mart is a multipurpose HTML Bootstrap template with a solid design specially crafted to help you built Business websites with a huge variety of options.

This Template is suitable for all type of Business related website. This template built using Bootstrap v5 Framework, Google Fonts, Font Awesome Free 5.15.2, Particle js v2.0.0, wow js, ScrollPos-Styler v0.7.1 . It is highly recomended that you are familiar in a way with at least HTML5 / CSS3 .

So let's jump directly to it, and start everything from scratch.

Downloading Biz Mart

Once you download the zip file, unzip it. You should have a folder with the following folder & files by now.

Templates css, fonts, images,scss, js

Installation

Install the template via ftp :

  1. Connect to your website host, using some of many FTP clients, for example FileZilla.
  2. Go to your default server folder, and copy Biz Mart into that folder.

Setting your fonts

To use google fonts, head over to Google's Fonts and add whichever font you want to your collection. Once you finished, follow the instructions and place the generated link in your HTML's file <head> tag.
The link should look like this : 


<link
rel="stylesheet"
href="https://fonts.googleapis.com/css?family=Poppins:300,400,500,600&display=swap"
type="text/css"
/>

For customize, go to style.css (css/style.css), search for the Typography table of content and replace the font family, with the font-family you just copied. 

 
font-family: "Poppins", sans-serif;

For customize, go to scss/config , search for the Typography table of content and replace the font family, with the font-family you just copied. 

 
font-family: "Poppins", sans-serif;

Changing favicon

Favicons are located in images folder. You can simply change them by replacing it with your own favicon icon, and include this code to the HTML Documents: 

      
        <link
        rel="apple-touch-icon"
        sizes="180x180"
        href="img/favicon/apple-touch-icon.png"
      />
      <link
        rel="icon"
        type="image/png"
        sizes="32x32"
        href="img/favicon/favicon-32x32.png"
      />
      <link
        rel="icon"
        type="image/png"
        sizes="16x16"
        href="img/favicon/favicon-16x16.png"
      />
      <link rel="manifest" href="img/favicon/site.webmanifest" />
Recommended service to use - Generated by RealFaviconGenerator or Favicon & App Icon Generator

Changing logo

To change logo, simply replace the logo.png at imag/logo/logo.png.

Please make sure that your logo size is 160 x 60 px and in png format .
